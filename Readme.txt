# MLOps задание №3

Задача: применить airflow и mlflow в задаче машинного обучения

## Комментарии к решению

В данной задаче я загружаю датасет с платформы kaggle.com с помощью download.py, провожу предобработку данных через preprocess.py, тренирую модель через train_model.py и, наконец, оцениваю полученный результат через evaluate.py.

Была выбрана задача предсказания цены, для решения была использована модель XGBoost. Все использованные в экспериментах параметры и метрики сохранены в mlflow, ниже приложены скриншоты с web ui.
